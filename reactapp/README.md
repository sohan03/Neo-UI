# UI

## Description

React-based user interface designed to provide an intuitive and seamless experience for managing events, orders, and user accounts. It features a responsive sidebar for navigation, dropdown menus for user options, and a visually appealing layout.

## Features

- **Responsive Design**: The sidebar collapses for mobile views, enhancing user experience on all devices.
- **Dropdown Menu**: Easily access account settings, privacy policy, and logout options through a convenient dropdown.
- **Navigation Links**: Quick access to key areas such as Home, Events, Orders, Settings, Change Log, and Support.
- **Upcoming Events Section**: Displays a list of upcoming events dynamically.
- **User Profile Section**: Personalized user interface with avatar and account details.
